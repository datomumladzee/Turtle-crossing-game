

# 🐢 Turtle Crossing Game

A simple arcade-style game built with Python using the **Turtle graphics** module.
The goal is to cross the road while avoiding moving cars. The project follows **OOP principles** with a modular structure.

---

## 🎮 Controls

* **↑ Up Arrow** – Move forward
* **↓ Down Arrow** – Move backward
* **R** – Restart game

---

## 🕹️ Gameplay

* Cross the road without hitting cars.
* Each successful crossing increases the level and car speed.
* Collision = Game Over. Press **R** to restart.

---

## 🧱 Project Files

* `main.py` – Game loop
* `player.py` – Player movement
* `car_manager.py` – Car generation & speed
* `scoreboard.py` – Level display & Game Over
* `game_manager.py` – Game state control

---

## 🚀 Run the Game

Make sure Python 3 is installed, then run:

```bash
python main.py
```
